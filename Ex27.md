## The Truth Tables

| NOT (!) | true?  |
|:-------:|:------:| 
| !false  | true   |
| !true   | false  |

|    AND (&&)    | true? |
|:--------------:|:-----:|
|  true && false | false |
|  true && true  |  true |
|  false && true | false |
| false && false | false |
 
|       OR       | true? |
|:--------------:|:-----:|
|  true or false |  true |
|  true or true  |  true |
|  false or true |  true |
| false or false | false |

NB. *or can also be written ||*

|   !=   | true? |
|:------:|:-----:|
| 1 != 0 |  true |
| 1 != 1 | false |
| 0 != 1 |  true |
| 0 != 0 | false |

|   ==   | true? |
|:------:|:-----:|
| 1 == 0 | false |
| 1 == 1 |  true |
| 0 == 1 | false |
| 0 == 0 |  true |
