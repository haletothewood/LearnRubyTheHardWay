## The Truth Tables

| NOT (!) | true?  |
|:-------:|:------:| 
| !false  | true   |
| !true   | false  |

|     OR (||)    | true? |
|:--------------:|:-----:|
|  true || false |  true |
|  true || true  |  true |
|  false || true |  true |
| false || false | false |

|    AND (&&)    | true? |
|:--------------:|:-----:|
|  true && false | false |
|  true && true  |  true |
|  false && true | false |
| false && false | false |

|   !=   | true? |
|:------:|:-----:|
| 1 != 0 |  true |
| 1 != 1 | false |
| 0 != 1 |  true |
| 0 != 0 | false |

|   ==   | true? |
|:------:|:-----:|
| 1 == 0 | false |
| 1 == 1 |  true |
| 0 == 1 | false |
| 0 == 0 |  true |
